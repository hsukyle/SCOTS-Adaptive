Writes, should they exist, a sequence of controller and controller domain BDDs to directories 'C' and 'Z' respectively that satisfy the reachability specification.

Parameters
[in]startAbs0-index of the abstraction to start with.
[in]minToGoCoarserMinimum number of growing fixed point iterations needed before an attempt to go to a coarser abstraction.
[in]minToBeValidMinimum number of growing fixed point iterations needed before a controller is declared valid.
[in]earlyBreakIf 1, the synthesis ends as soon as I meets the domain of C.
[in]verboseIf 1, prints additional information during synthesis to the log file.
Returns
1 if controller(s) satisfying specification is/are synthesized; 0 otherwise.
