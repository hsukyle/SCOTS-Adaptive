Detailed Description

template<class X_type, class U_type>
class scots::Adaptive< X_type, U_type >

A class that does abstraction-based synthesis with adaptive time- and state space-gridding.

◆ alwaysEventually()

template<class X_type , class U_type >
int scots::Adaptive< X_type, U_type >::alwaysEventually ( int  startAbs,
int  minToGoCoarser,
int  minToBeValid,
int  verbose = 0,
int  earlyBreak = 0 
)
inline

Writes, should they exist, a sequence of controller and controller domain BDDs to directories 'C' and 'Z' respectively that satisfy the Buchi box-diamond (aka always-eventually) specification. Note: of the n resultant controllers in 'C', the n-1th, .., 1st controllers are to reach, and the nth controller is for when the system state is a goal state.

Parameters
[in]startAbs0-index of the abstraction to start with.
[in]minToGoCoarserMinimum number of growing fixed point iterations needed before an attempt to go to a coarser abstraction.
[in]minToBeValidMinimum number of growing fixed point iterations needed before a controller is declared valid.
[in]verboseIf 1, prints additional information during synthesis to the console.
[in]earlyBreakIf 1, reachability ends as soon as I meets the domain of C.
Returns
1 if controller(s) satisfying specification is/are synthesized; 0 otherwise.
